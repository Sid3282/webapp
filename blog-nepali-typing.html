Mastering Nepali Typing: Unicode vs Preeti

By Hamro Utility Team Posted on April 15, 2023 6 min read

Nepali typing has evolved significantly with the introduction of Unicode standards, but many still use the traditional Preeti font. This guide explains the differences between these systems and provides practical tips for efficient Nepali typing in both formats.

Unicode vs Preeti: Key Differences

Feature Unicode Preeti
Standard International standard Nepal-specific
Compatibility Works everywhere Requires font installation
Typing Method Phonetic (Roman to Nepali) Direct key mapping
Future-proof Yes No
Example नेपाल cf]nf

Recommendation:

For long-term compatibility and ease of use, we recommend switching to Unicode Nepali typing. Most modern devices and platforms support it natively.

Tips and Tricks for Nepali Unicode Typing

1. Combining English with Nepali Text

Simply put any English text inside curly {} brackets to keep it in English:

Input: yo {mobile} mero ho

Output: यो mobile मेरो हो

2. Phonetically Similar Letters

Some Nepali letters have case-sensitive mappings:

  • ta = त, Ta = ट
  • tha = थ, Tha = ठ
  • da = द, Da = ड
  • dha = ध, Dha = ढ
  • na = न, Na = ण
  • sha = श, Sha = ष

For other Nepali letters, the upper or lower case doesn't matter.

3. Handling Letter Combinations

If a Nepali letter gets wrongly combined with the preceding letter, use the slash (/) key to separate:

Input: pratishatko

Output: प्रतिशत्को

Input: pratishat/ko

Output: प्रतिशतको

4. Adding Halanta (्)

Use backslash (\) at the end of a word to add halanta:

Input: bas

Output: बस

Input: bas\

Output: बस्

5. Special Characters and Combinations

  • ri^ = रि (as in प्र)
  • rr = र्‍ (as in गर्‍य)
  • rri = ऋ
  • rree = ॠ
  • yna = ञ
  • chha = छ
  • ksha = क्ष
  • gya = ज्ञ
  • * = अनुस्वर (ं)
  • ** = चन्द्रबिन्दु (ँ)
  • om = ॐ

Preeti Typing Cheat Sheet

For those still using Preeti font, here are some common character mappings:

a = अ, b = ब, c = च, d = द, e = ए, f = फ, g = ग, h = ह, i = इ, j = ज

k = क, l = ल, m = म, n = न, o = ओ, p = प, q = अ, r = र, s = स, t = त

u = उ, v = भ, w = व, x = छ, y = य, z = झ

Converting Between Unicode and Preeti

If you need to convert text between these formats:

  1. Use online conversion tools (many are available)
  2. Install font conversion software
  3. For documents, use "Find and Replace" with character mapping

Pro Tip:

When typing in Unicode, use a proper Nepali keyboard layout or input method editor (IME) for best results. Many are available as browser extensions or system utilities.

Conclusion

Whether you choose Unicode or Preeti depends on your needs, but Unicode is clearly the future of Nepali digital content. By mastering these typing techniques, you'll be able to communicate effectively in Nepali across all digital platforms. Practice regularly and refer to this guide whenever you encounter typing challenges.
